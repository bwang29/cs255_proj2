hi,

try something like this:

    Certificate certificate = keystore.getCertificate(alias);
    iaik.x509.X509Certificate iaikCertificate;
    if (certificate instanceof iaik.x509.X509Certificate) {
        iaikCertificate = (iaik.x509.X509Certificate) certificate;
    } else {
        iaikCertificate = new
iaik.x509.X509Certificate(certificate.getEncoded())
    }

you can also use a java.security.CertificateFactory to create the
certificate.

    CertificateFactory factory = CertificateFactory.getInstance("X.509",
"IAIK");
    InputStream certificateInputStream = new
java.io.ByteArrayInputStream(certificate.getEncoded());
    iaikCertificate = (iaik.x509.X509Certificate)
factory.generateCertificate(certificateInputStream);
